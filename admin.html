GitHub — Publicação automática
Para publicar posts automaticamente no site, configure seu token do GitHub. Os posts são salvos no arquivo posts.json do repositório.
Como obter o token:
1. Acesse github.com → Settings → Developer settings → Personal access tokens
2. Clique em "Generate new token (classic)"
3. Marque a permissão repo
4. Copie e cole o token abaixo
